# 🍕 Fast Pizza

A modern and responsive pizza ordering application built with React, Redux Toolkit, React Router, and Tailwind CSS.

Fast Pizza allows users to browse a menu, add items to a cart, place orders, track order status, and optionally prioritize deliveries. The application provides a smooth ordering experience with geolocation-powered address detection and a mobile-first user interface.

---

## ✨ Features

### Menu & Ordering

- Browse available pizzas
- Add items to cart
- Update item quantities
- Remove items from cart
- Clear cart functionality
- Create and submit orders

### Order Management

- Track orders by ID
- View order details and status
- Upgrade existing orders to priority delivery

### User Experience

- Responsive design for all screen sizes
- Geolocation-based address detection
- Form validation and error handling
- Loading indicators and feedback states
- Modern UI with improved accessibility

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/adenijimarvellous/fast-pizza.git
```

Navigate into the project:

```bash
cd fast-pizza
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
